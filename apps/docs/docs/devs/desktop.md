# Desktop App

The Chaos Reactor desktop app is build with [Tauri](https://tauri.app/).

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/en/) (v14.15.1)
- [Rust](https://www.rust-lang.org/tools/install) (v1.49.0)
- [NPM](https://www.npmjs.com/get-npm)
